# ЗдравГарант — консольная медицинская система

Консольное приложение на Python, реализующее пользовательские сценарии
медицинской системы «ЗдравГарант»: регистрация, вход, запись на диагностику,
управление записями, редактирование профиля, сохранение и загрузка данных.

## Архитектура

Проект построен на принципах ООП с разделением ответственности:

| Модуль | Класс | Роль |
|---|---|---|
| `person.py` | `Person` | Базовый класс (id, ФИО, возраст) |
| `patient.py` | `Patient(Person)` | Пациент (пароль, список записей) |
| `doctor.py` | `Doctor(Person)` | Врач (специализация, клиника) |
| `clinic.py` | `Clinic` | Клиника (название, отделение) |
| `appointment.py` | `Appointment` | Запись на диагностику |
| `console_io.py` | `ConsoleIO` | Стратегия ввода/вывода |
| `storage.py` | `PickleStorage` | Стратегия хранения (pickle) |
| `medical_system.py` | `MedicalSystem` | Контейнер — бизнес-логика |
| `main.py` | — | Точка входа, текстовое меню |

### Ключевые решения

- **Наследование**: `Patient` и `Doctor` наследуют `Person`.
- **Стратегия ввода/вывода**: весь консольный ввод/вывод проходит через `ConsoleIO`.
- **Стратегия хранения**: сериализация через `PickleStorage` (модуль `pickle`).
- **Контейнерный класс**: `MedicalSystem` агрегирует все сущности.
- **Меню через словарь функций**: обработка выбора без `if-elif` цепочек.

## Список классов

- `Person` — базовый класс с полями `id`, `full_name`, `age`.
- `Patient(Person)` — добавляет `password`, `appointments`.
- `Doctor(Person)` — добавляет `specialization`, `clinic_id`.
- `Clinic` — поля `clinic_id`, `clinic_name`, `department`.
- `Appointment` — поля `appointment_id`, `patient_id`, `doctor_id`, `clinic_id`, `date`, `status`.
- `ConsoleIO` — методы `input_str`, `input_int`, `display`, `display_list`, `show_menu`, `input_choice`.
- `PickleStorage` — методы `save`, `load`.
- `MedicalSystem` — хранит списки сущностей, реализует всю бизнес-логику.

## Запуск

```bash
python main.py
```

Требуется Python 3.10+ (стандартная библиотека, дополнительные пакеты не нужны).
